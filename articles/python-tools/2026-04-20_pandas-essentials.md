### GroupBy Power
```python
df.groupby('dept').agg(
    avg_salary=('salary', 'mean'),
    headcount=('employee_id', 'nunique'),
    total_sales=('sales', 'sum')
)

df['salary_zscore'] = df.groupby('dept')['salary'].transform(
    lambda x: (x - x.mean()) / x.std()
)
```
